# HypeSquad-Badge-Claim-Script

Requirements:

A modern web browser (Google Chrome or Mozilla Firefox recommended).

An active session logged into the Discord web application at discord.com/app.

Usage Instructions:

  Open the Discord web application in your browser.

Access the Developer Console:

                Windows / Linux: Press Ctrl + Shift + J

                Mac: Press Cmd + Option + J

                Alternatively, right-click anywhere on the page, select Inspect, then navigate to the Console tab.

Locate the house_id variable within the script and set it to your preferred HypeSquad house value (1, 2, or 3).

Copy the complete script, paste it into the Console, and press Enter to execute.
